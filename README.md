# Galería de Imágenes en JavaScript

Este proyecto contiene una serie de ejemplos de galerías de imágenes implementadas en HTML y JavaScript. Cada archivo muestra una técnica diferente para manipular imágenes y eventos en la web.

```

## Descripción de los Archivos

### [01_galeria.html](01_galeria.html)
Demuestra cómo cambiar la imagen principal usando botones y el evento `onclick` o `onmouseover`. Cambia el borde de la imagen según la foto seleccionada.

### [02-galeria.html](02-galeria.html)
Similar al anterior, pero añade lógica para cambiar el color y grosor del borde dependiendo de la imagen seleccionada, usando condicionales `if-else`.

### [03-galeria.html](03-galeria.html)
Utiliza una estructura `switch` para cambiar el color del borde de la imagen según la foto seleccionada. Es una variante del ejemplo anterior, mostrando otra forma de controlar los estilos.

### [04-galeria.html](04-galeria.html)
Ejemplo básico de cambio de imagen usando botones y eventos. Incluye un botón que usa `onmouseover` para cambiar la imagen al pasar el ratón.

### [05-galeria-for.html](05-galeria-for.html)
Genera dinámicamente una lista de números usando un bucle `for` y los muestra en la página. No cambia imágenes, solo muestra cómo usar bucles para generar contenido HTML.

### [05-galeria-for_01.html](05-galeria-for_01.html)
Similar al anterior, pero aquí sí se generan botones dinámicamente para cambiar la imagen principal al hacer clic, usando un bucle `for` y concatenando el nombre de la imagen.

### [06-galeria-array.html](06-galeria-array.html)
Introduce el uso de arrays para manejar la galería de imágenes. Genera los botones de forma dinámica a partir de un array de nombres de archivos de imagen, facilitando la gestión de la galería.

### [07-galeria-array.html](07-galeria-array.html)
Extiende el ejemplo anterior añadiendo arrays para los títulos y los colores del borde de cada imagen. Al seleccionar una imagen, se actualiza el borde y el título dinámicamente.

### [08-galeria-array.html](08-galeria-array.html)
Similar al anterior, pero añade una botonera con estilos personalizados y resalta el botón de la imagen seleccionada. También muestra cómo actualizar el título de la imagen.

### [09-galeria-timer.html](09-galeria-timer.html)
Implementa una galería automática que cambia de imagen cada segundo usando `setInterval`. Incluye un botón para detener la rotación automática de imágenes.

### Carpeta [img/](img/)
Contiene todas las imágenes utilizadas en los ejemplos de la galería.

## Cómo usar

Abre cualquiera de los archivos `.html` en tu navegador para ver el ejemplo correspondiente. Cada archivo es independiente y muestra una técnica diferente para crear galerías de imágenes interactivas con JavaScript.
